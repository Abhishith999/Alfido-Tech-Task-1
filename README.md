# Reconnaissance & Vulnerability Scanning

This project demonstrates active reconnaissance and vulnerability assessment on safe test systems using industry-standard cybersecurity tools such as Nmap and Nikto.

## Project Objectives

* Perform network reconnaissance
* Enumerate open ports and running services
* Identify security vulnerabilities
* Analyze security risks and misconfigurations
* Document findings professionally

## Tools Used

* Nmap
* Nikto
* Kali Linux

## Tasks Performed

### Nmap Scanning

* Service version detection
* OS detection
* Port enumeration
* Service identification

### Nikto Vulnerability Scanning

* Web server security analysis
* Detection of missing security headers
* Identification of exposed HTTP methods
* Information disclosure detection

## Key Findings

* Open SSH and HTTP services detected
* Apache and IIS web servers identified
* Missing security headers discovered
* Internal IP disclosure vulnerability found
* HTTP TRACE method enabled
* Multiple medium-risk security issues identified

## Deliverables

* Scan result screenshots
* Professional vulnerability assessment report
* Security findings and recommendations

## Learning Outcomes

This project provided practical experience in:

* Network reconnaissance
* Vulnerability assessment
* Security analysis
* Risk identification
* Cybersecurity documentation and reporting

## Disclaimer

This assessment was performed only on authorized and publicly available test systems intended for security testing and educational purposes.
